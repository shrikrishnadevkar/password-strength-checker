# Password Strength Checker (Python)

## Description
A beginner-friendly Python project that checks password strength according to basic cybersecurity standards.

## Features
- Checks minimum 8 characters
- Checks for numbers
- Checks for uppercase and lowercase letters
- Checks for special symbols
- Gives clear feedback to the user

## Technologies Used
- Python
- Regular Expressions (re module)
